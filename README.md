# Modern Geek News

## Getting Started
1. Install `Stylus` extension/add-on
  - Chrome: [Stylus](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
  - Firefox: [Stylus](https://addons.mozilla.org/en-US/firefox/addon/styl-us/)

2. Add `modern-geek-news.css` or install the style from the [Userworld](https://userstyles.world/style/27566)
